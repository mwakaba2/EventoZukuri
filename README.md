# Evento Zukuri #
## Meet-up Event Planner ##
Mobile first web app for creating events!

Udacity Senior Developer Nanodegree Project One

### App Features ###
  * User Account Creation
  * Event Creation
  * Read access to Events

### To run the application ###
```bash
gulp serve
```
### Tech Stack ###
  * Angular
  * Firebase
  * Gulp
  * Yeoman
  * Bower
